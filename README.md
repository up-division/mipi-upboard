# mipi-upboard
This is a simple demonstration of the v4l2 device, it's show you how the MIPI camera works on the UP platform using the Linux kernel native Intel IPU driver.
For support of the Intel PSYS driver, please contact your camera manufacturer or us at https://up-board.org/contact-us/

Supported products
=============================================
* UP Xtreme i12
* UP Xtreme i14
* UP Xtreme ARL
* UP Squared Pro 7000
* UP Squared i12

Supported camera sensors
=============================================
* AR0234
* OV5647
* IMX219
* IMX477
* IMX708
  
Special Requirements
=============================================
linux kernel 6.14+, following Intel IPU6 driver from kernel mainline.

Content of this repository
=============================================
* mipi camera demo script
* camera sensor drivers
* Intel ipu6 driver
